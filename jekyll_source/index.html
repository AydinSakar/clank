Clank

An open source HTML/CSS framework for prototyping mobile iOS and Android apps

Watch the Clank intro to get a good idea of what Clank is and what it does.

Running time: 02:48

Open source

Clank is free and open source and it will remain so. We welcome improvements and suggestions on Github.

Component based

Pick and choose. Just like known frameworks like Twitter Bootstrap, pick what you need and use it.

Modern SCSS

The Clank code is written using modern CSS techniques. Under the hood it utilizes Sass, Compass and BEM style coding to ensure a maintainable codebase.

As automatic as possible

Use generators to quickly create new pages. Automatically create icon fonts from vector files.
