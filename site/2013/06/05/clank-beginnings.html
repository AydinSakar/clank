Clank - Beginnings

Posted on June 05, 2013 by Wolfr

This is the blog for the Clank open source project.

You might be wondering: WTF is Clank? Clank is an open source HTML/CSS framework for prototyping native apps. It’s a set of components that will help you communicate your design.

In theory you could use Clank to make a real app – but I wouldn’t recommend it. Browser support is minimal as Clank is used to convey the idea of an app to app developers.

I am looking for contributors to help with the actual website, documentation and testing the framework. Get in touch.
